### Skills 基础

**核心概念** ： Skill = “一个能被 Agent 自动调用的独立能力包”。 它可以是浏览器自动填表、邮件智能归档、航班自动值机、甚至让 AI 自己帮你写新 Skill。

一句话： **没有 Skill，OpenClaw 只是个高级聊天机器人；有了 Skill，它才是真正的数字分身** 。

#### 5.1 三类 Skills（必须分清）

| 类型        | 安装方式         | 位置                             | 适合人群       | 安全性          |
| --------- | ------------ | ------------------------------ | ---------- | ------------ |
| Bundled   | 官方内置         | 随 Gateway 安装                   | 所有人        | ★★★★★        |
| Managed   | ClawHub 一键安装 | ~/.openclaw/skills/managed/   | 普通用户       | ★★★★☆        |
| Workspace | 自己写或 fork    | ~/.openclaw/workspace/skills/ | 开发者 / 高级玩家 | ★★★★★（你自己控制）

**Bundled** （官方自带，永远安全）：

-   browser_control、filesystem、terminal、email、calendar 等 28 个

**Managed** （ClawHub 市场）：

-   社区上传，需审核，但仍有风险（2026 年已下架 400+ 恶意 Skill）

**Workspace** （本地自写）：

-   你完全掌控，推荐所有高危需求都写在这里

```
#!/usr/bin/env python3
# /// script
# requires-python = ">=3.10"
# dependencies = ["pyzotero>=1.6.0"]
# ///

import argparse
import os
import sys
from pyzotero import zotero

def main():
    parser = argparse.ArgumentParser(description="保存论文到 Zotero 文献库")
    parser.add_argument("--title", required=True, help="论文标题")
    parser.add_argument("--authors", required=True, help="作者列表，以逗号分隔")
    parser.add_argument("--url", required=True, help="论文链接（用于查重）")
    parser.add_argument("--abstract", help="论文摘要")
    parser.add_argument("--summary", help="AI 生成的摘要")
    parser.add_argument("--tags", help="标签列表，以逗号分隔")

    args = parser.parse_args()

    library_id = None
    api_key = None

    zotero_creds = os.environ.get('ZOTERO_CREDENTIALS')
    if zotero_creds and ':' in zotero_creds:
        try:
            parts = zotero_creds.strip().split(':')
            if len(parts) == 2:
                library_id = parts[0].strip()
                api_key = parts[1].strip()
        except:
            pass

    library_type = 'user' # 默认使用个人文献库

    if not library_id or not api_key:
        print("错误：需要 Zotero 凭据。", file=sys.stderr)
        print("请设置 ZOTERO_CREDENTIALS='userID:apiKey'（用于 UI 配置）", file=sys.stderr)
        sys.exit(1)

    try:
        zot = zotero.Zotero(library_id, library_type, api_key)

        # 通过 URL 检查是否已存在
        # 使用搜索查询匹配 URL
        print(f"正在检查是否已存在该条目：{args.url}...")
        items = zot.items(q=args.url, limit=1)

        if items:
            print(f"该条目已存在：{items[0].get('data', {}).get('title', '未知')}")
            # 暂时跳过更新，避免覆盖用户数据
            print(f"Zotero 链接：https://www.zotero.org/{library_id}/items/{items[0]['key']}")
            return

        # 创建新条目
        template = zot.item_template('journalArticle')
        template['title'] = args.title

        # 格式化作者信息
        creators = []
        for author in args.authors.split(','):
            name_parts = author.strip().split(' ')
            if len(name_parts) > 1:
                creators.append({'creatorType': 'author', 'firstName': ' '.join(name_parts[:-1]), 'lastName': name_parts[-1]})
            else:
                creators.append({'creatorType': 'author', 'lastName': author.strip(), 'firstName': ''})
        template['creators'] = creators

        template['url'] = args.url
        if args.abstract:
            template['abstractNote'] = args.abstract

        # 添加标签
        if args.tags:
            template['tags'] = [{'tag': t.strip()} for t in args.tags.split(',')]

        print(f"正在创建条目：{args.title}...")
        resp = zot.create_items([template])

        if resp.get('successful'):
            new_item = resp['successful']['0']
            item_key = new_item['key']
            print(f"条目创建成功！")

            # 将摘要添加为笔记
            if args.summary:
                print("正在添加摘要笔记...")
                note_template = zot.item_template('note')
                note_template['note'] = f"<h3>AI 摘要</h3><p>{args.summary}</p>"
                note_template['parentItem'] = item_key
                zot.create_items([note_template])
                print("笔记已添加。")

            # 下载并附加 PDF
            if 'arxiv.org' in args.url:
                try:
                    import urllib.request
                    import tempfile

                    # 将摘要链接转换为 PDF 链接
                    pdf_url = args.url.replace('/abs/', '/pdf/')
                    if not pdf_url.endswith('.pdf'):
                        pdf_url += '.pdf'

                    print(f"正在下载 PDF...")

                    # 设置 User-Agent 以支持下载
                    opener = urllib.request.build_opener()
                    opener.addheaders = [('User-agent', 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7) Chrome/120.0.0.0')]
                    urllib.request.install_opener(opener)

                    # 创建安全的文件名
                    safe_title = "".join(c for c in args.title if c.isalnum() or c in (" ", "-", "_")).strip()
                    safe_title = safe_title[:50] # 限制长度
                    safe_filename = f"{safe_title}.pdf"

                    # 使用临时目录，但指定文件名
                    with tempfile.TemporaryDirectory() as td:
                        pdf_path = os.path.join(td, safe_filename)
                        urllib.request.urlretrieve(pdf_url, pdf_path)

                        print(f"正在上传 PDF 附件（{safe_filename}）...")
                        zot.attachment_simple([pdf_path], item_key)
                        print("PDF 已附加。")

                except Exception as e:
                    print(f"附加 PDF 失败：{e}", file=sys.stderr)

            print("注意: 请在 Zotero 客户端点击 '同步' (Sync) 按钮以获取云端 PDF 文件。")
            print("         如果是 Web API 写入，本地客户端必须同步才能看到附件内容。")

        else:
            print(f"创建条目失败：{resp}", file=sys.stderr)
            sys.exit(1)

    except Exception as e:
        print(f"错误：{e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    main()
```

#### 5.3 新手必装10个 Skill（按优先级排序）

这份清单主打低风险、高实用、立竿见影。来源主要是 @steipete 等靠谱作者，安装量高、Star 多、恶意报告极少。

```
clawhub install <slug>

npx clawhub@latest install <slug>

**第一优先级：安全基础**

1.  **skill-vetter（安全扫描）**

社区共识："后悔没先装"。这是你安装任何其他 Skill 之前必须先装的，它能扫描其他 Skill 的代码，检测潜在的恶意行为。

```
clawhub install skill-vetter
```

先有了这个，你才能放心地去装其他东西。

**第二优先级：联网能力**

2.  **tavily-search（联网搜索）**

没有这个，你的 Agent 就是个"井底之蛙"——它只有训练数据里的知识，查不了任何实时信息。这几乎是所有新手第一梯队必装的 Skill，37k+ 安装量，AI Agent 的标配。

```
clawhub install tavily-search
```

**第三优先级：让它变聪明**

3.  **self-improving-agent（自我迭代）**

ClawHub 热门榜第一，46k+ 安装量。让 Agent 记住自己的错误、自我优化、越来越聪明。新手装了这个，最容易感受到"哇，它好像变聪明了"的直观反馈。

```
clawhub install self-improving-agent
```

4.  **proactive-agent（主动性）**

让 Agent 从"被动回答问题"变成"主动帮你规划和推进任务"。中文社区里装了这个的用户普遍反馈："感觉它活了。" 这个 Skill 是实现"数字员工"体验的关键一步。

```
clawhub install proactive-agent
```

**第四优先级：日常生产力**

5.  **gog（Google Workspace CLI）**

Gmail、Google 日历、Google Drive、Google Docs 全家桶一次打通，46k+ 安装量。如果你是 Google 系用户，这是你能最快看到自动化效果的 Skill——读邮件、加日历、写文档，全部可以自动化。

```
clawhub install gog
```

6.  **github（GitHub 集成）**

能搜代码、管 issue/PR、创建 repo，35k+ 安装量。开发者入门必备，让你的 Agent 能直接参与代码仓库的管理，不再需要手动在网页上操作。

```
clawhub install github
```

**第五优先级：信息处理**

7.  **summarize（内容总结）**

支持总结 URL、PDF、图片、YouTube 视频、音频文件，36k+ 安装量，高频使用。研究东西的时候超级省力——不需要你一个字一个字去读，扔给它，几秒钟给你核心要点。

```
clawhub install summarize
```

8.  **find-skills（技能发现）**

让 Agent 自己去 ClawHub 上搜索并推荐适合你的 Skill，解决"不知道该装什么"的最大痛点。这是一个"元技能"——用来发现其他技能的技能，新手最友好。

```
clawhub install find-skills
```

**第六优先级：长期记忆**

9.  **ontology / memory（结构化记忆）**

让 Agent 真正"记住你"，跨对话保持连贯，不再每次都从零开始。35k+ 安装量，短期看不出太大差距，但长期使用越用越香——它对你的了解会越来越深，服务越来越精准。

```
clawhub install ontology

#####  四个基础知识管理 Skill

如果你有知识管理需求，以下四个 Skill 构成了一条完整的信息处理链路：

**Multi Search Engine**：覆盖17个国内外搜索引擎，让 OpenClaw 能真正在国内网络环境下正常搜索。

```
npx clawhub@latest install multi-search-engine
```

**x-reader**：支持解析微信公众号、小红书、B站、X 等国内主流平台的链接内容。

```
pip install git+https://github.com/runesleo/x-reader.git
```

**Obsidian**：让 OpenClaw 能直接往你的 Obsidian 知识库里存内容，打通"信息输入"和"知识沉淀"的通道。

```
npx clawhub@latest install obsidian
```

**find-skills**：搜索和发现更多可用的 Skill，保持能力持续扩展。

```
npx clawhub@latest install find-skills
```

装完这四个，OpenClaw 具备了"搜索 → 解析链接 → 存进知识库 → 发现新能力"的完整基础链路。
